## Instalación

Para clonar y ejecutar estos ejercicios, necesitarás [Git](https://git-scm.com) y [Node.js](https://nodejs.org/en/download/) (que viene con [npm](http://npmjs.com)) instalados en tu computadora. Desde tu línea de comandos:

```bash
# Clona este repositorio
$ git clone https://github.com/usuario/repositorio-curso-react

# Entra en el directorio del repositorio
$ cd repositorio-curso-react

# Instala las dependencias
$ npm install
```

## Estructura del Repositorio

Este repositorio está estructurado de manera que cada ejercicio de clase se encuentra en su propio directorio, con su propio conjunto de archivos de código fuente. Esto te permite concentrarte en un solo concepto a la vez.

```
.
├── Clase1
│   ├── Ejercicio1
│   └── Ejercicio2
├── Clase2
│   ├── Ejercicio1
│   └── Ejercicio2
└── ...
```

## Ejecutar los ejercicios

Para ejecutar los ejercicios en tu entorno local, necesitarás seguir los siguientes pasos:

1. Asegúrate de estar en la raíz del repositorio.
2. Navega a la carpeta del ejercicio que quieres ejecutar usando `cd`.
3. Ejecuta `npm install` para instalar las dependencias necesarias para ese proyecto (si es la primera vez que abres el proyecto).
4. Ejecuta `npm start` para iniciar el servidor de desarrollo.

```bash
# Ejemplo de cómo ejecutar el ejercicio 1 de la clase 1
$ cd Clase1/Ejercicio1
$ npm install
$ npm start
```

Luego, puedes abrir tu navegador y visitar `localhost:3000` para ver la aplicación en funcionamiento.

## Nota importante

No se incluyen las carpetas `node_modules` en este repositorio. Por lo tanto, **debes ejecutar `npm install` dentro de la carpeta de cada ejercicio antes de poder ejecutarlo**. Esto instalará todas las dependencias necesarias para el ejercicio.
